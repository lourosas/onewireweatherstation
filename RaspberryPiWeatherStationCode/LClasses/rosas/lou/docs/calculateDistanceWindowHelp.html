To Calculate the Distance of a given run:
1) Enter the Time of your run (Run Time) in Hours, Minutes and Seconds.
The seconds field can be entered in decimal format.
2) Enter the Pace of your run in Minutes and Seconds.
Again, the seconds field can be entered in decimal format.
3) Press the Calculate Distance button.

Press the Clear button to clear the data entered.

Press the Cancel button to cancel the calculation.

To save the data, press the Save button.

You can change the units by selecting Miles, Kilometers or Meters radio button.

NOTE: In order to calculate the Run Distance, data must be entered in the Minutes and Seconds text fields for both the Run Time and Pace entries. The Hours field in the Run Time can be left blank.
